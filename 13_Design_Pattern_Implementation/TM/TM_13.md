## Soal
Jelaskan dengan kemampuanmu apa itu event delegation dalam design pattern JavaScript. Tidak ada batas bobot kata dalam menjawab tugas ini, tetapi penilaian akan bergantung dari sepaham apa dan sebagus apa kamu menyajikan jawabanmu.


## Deskripsi
Event delegation itu bukan design pattern beneran kayak Singleton atau MVC Bedanya pattern–pattern tujuannya ngatur arsitektur kelas sementara event delegation cuma trik praktis buat nanganin event di DOM.
Konsepnya sederhana: cukup satu listener di elemen induk bukan banyak listener di tiap elemen anak memanfaatkan event bubbling jadi event dari anak bakal naik ke induk terus di induk tinggal cek event.target buat tahu anak mana yang diklik.
Efeknya memori lebih irit kode nggak berantakan dan elemen yang muncul belakangan (hasil manipulasi DOM dinamis) otomatis punya event handler tanpa perlu setup ulang.
